# MVA Trading App

Streamlit app for Monsoon Volatility Arbitrage strategy using Zerodha API.

## Deployment
- Deploy via [Streamlit Sharing](https://share.streamlit.io).
- Set secrets in Streamlit dashboard: GSHEET_ID, GCP_SERVICE_ACCOUNT_JSON, etc.

## Local Run
pip install -r requirements.txt
streamlit run mva_trading_app.py
